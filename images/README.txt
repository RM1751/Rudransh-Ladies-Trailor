========================================
HOW TO ADD YOUR IMAGES
========================================

STEP 1: Add Your Images Here
----------------------------
Copy your actual work photos to this folder.
Name them like this:

  blouse1.jpg
  blouse2.jpg
  kurti1.jpg
  kurti2.jpg
  salwar1.jpg
  lehenga1.jpg
  gown1.jpg

Supported formats: .jpg, .jpeg, .png

STEP 2: Update gallery.html
--------------------------
Open gallery.html and replace the sample image 
references with your actual image filenames.

Look for sections like:
  <img src="images/blouse1.jpg">

And update the src to match your image names.

STEP 3: Push to GitHub
----------------------
After adding images:
  git add .
  git commit -m "Added gallery images"
  git push

Then redeploy on Vercel/Netlify.

IMPORTANT NOTES:
----------------
- Images must be in this folder before pushing
- Keep image sizes under 1MB for faster loading
- Use descriptive names (e.g., princess-cut-blouse.jpg)
- The admin panel localStorage method only works 
  on your computer - use this folder method for 
  images to show on all devices!
